# 硬件连接

<p>
<img src="extras/Images/mini_gif_connection.gif" class="width50">
</p>

将 <span class="simple">Simple<span class="foc">FOC</span>Mini</span> 连接到微控制器、无刷直流电机和电源非常简单。

<span class="simple">Simple<span class="foc">FOC</span>Mini</span> 开发板有两个版本，v1.0 和 v1.1。这两个版本功能相同，但引脚顺序不同。<br>它们很容易区分：
- 你可以很容易地在开发板的丝印上看到版本标识。
- v1.1 在排针的第一行有两个 GND 引脚，而 v1.0 在排针的第一行只有一个 GND 引脚。
